## 🧩 What is Perspective-Studio?

Perspective-Studio is an app designed to run open-source AI models on your Mac. It runs locally, which means your data stays on your device. You interact with it through a clean, simple SwiftUI interface made specifically for Apple machines. This tool lets you explore large language models (LLMs) and machine learning without needing cloud services or programming skills.

## 🎯 Key Features

- Run multiple AI models directly on your Mac  
- Easy-to-use interface built with SwiftUI  
- Supports Apple Silicon for better speed and efficiency  
- Works offline; no internet connection required while running models  
- Accessibility support, including VoiceOver for visual assistance  
- Text generation and voice interaction ready  
- Compatible with popular transformers and HuggingFace models

## 📦 How to Use Perspective-Studio

After installing, open Perspective-Studio from your Applications folder or Launchpad.

1. **Choose a Model:** The interface will show a list of AI models available to run locally. Pick the one you want to try.

2. **Load the Model:** Tap the load button. The app will prepare the model on your Mac. This might take a few moments.

3. **Interact:** Start typing your queries or prompts in the text box. Press Enter or click send.

4. **Get Answers:** The AI model processes your input and replies within the app’s window.

5. **Voice Control:** You can use voice commands if your system supports VoiceOver and speech input.

## 🛠 Troubleshooting

If the app does not start or crashes:

- Make sure your macOS version meets the stated requirements.  
- Verify the app is from the official GitHub release page.  
- Restart your Mac and try opening the app again.  
- Close other heavy apps to free up memory before running Perspective-Studio.  
- Check your macOS security settings under System Preferences > Security & Privacy, allowing apps from identified developers.

For model loading issues:

- Confirm you have enough free disk space.  
- Try switching to a smaller AI model from the selection screen.
